Features

SpreadTable Version Angular Version
4.* 14
3.* 13
  • ✅ virtual scroll for displaying big data super fast
  • ✅ copy/paste/cut functionality with keyboard shortcuts
  • ✅ copy/paste/cut in bulk
  • ✅ copy/paste/cut from and into excel
  • ✅ custom validators support
  • ✅ custom renderer and editor
  • ✅ disabled columns
  • ✅ undo/redo on single or batch changes
  • ✅ resizable columns
  • ✅ extensible column/context menus with events
